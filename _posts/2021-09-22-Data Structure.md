## Pointer
👉 자료형과 주소값을 저장하는 변수
👉 int *ptr = &a; // & ; 주소 연산자 (ampersand)
<br>
👋 포인터 변수 앞 * 붙이면 해당 주소에 저장된 값을 의미
👋 int a = 3; (가정) // cout << *ptr << endl; // 3 출력

<br>

## Call-by-Value VS Call-by-Reference ✔️
### Call-by-Value
👉 함수 호출 시 인자 전달 과정에서 발생
👉 데이터를 복사하는 과정에서 발생

### Call-by-Reference
👉 전달되는 인자를 레퍼런스로 받으면 데이터의 복사 연산이 필요하지 않음
#### 👉 ∵ 성능 향상
👉 원본 데이터 변경 막을때는 const
